# vue-project

Vue.js

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test

#Functional description
1. View a list of articles;
2. Added pagination;
3. Ability to go to a specific page;
4. Ability to navigate through the pages using the buttons pagination "forward" and "back";
5. View a specific article.
6. Returning from the article to the page with the list of articles;
7. Deleting an article (if you add a token to the code src/store/actions/services,js value "accessToken", this functional is works);
8. Mobile friendly
9. Added unit tests with mock

```
